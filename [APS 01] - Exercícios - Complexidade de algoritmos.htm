Texto da questão

Para um dado problema, se a complexidade do melhor caso de um algoritmo é: O(nlogn)O(nlog⁡n), então o limite inferior deste problema é Ω(nlogn)Ω(nlog⁡n). Esta afirmação é correta? 

Escolha uma opção:

Feedback

A complexidade no melhor caso mede a situação em que ocorre uma entrada específica que exigirá o mínimo de execuções da operação fundamental, ou seja, o mínimo de esforço do computador. Se neste caso, ao medir a complexidade do algoritmo temos uma ordem n log nn log n, significa que esta ordem não pode ser reduzida para uma função de menor ordem. Como a notação OO indica um limite superior significa que temos uma relação de <=<=, ou seja, existe uma faixa de valores "menores ou igual". Como temos uma ordem O(n log n)O(n log n) e o melhor caso foi O(n log n)O(n log n) temos uma relação de igualdade e portanto podemos concluir que a afirmação é verdadeira, e portanto o limite inferior do algoritmo é Ω(n log n)Ω(n log n) uma vez que não conseguiremos obter uma função de menor ordem.

A resposta correta é 'Verdadeiro'.
